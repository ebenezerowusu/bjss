BJSS secures fourth AWS Competency – this time for cloud security

Written by Chloe Coggins | Dec 13, 2021 9:21:40 AM

The Amazon Web Services (AWS) Competency Program is designed to identify, validate, and promote AWS Partners with demonstrated AWS technical expertise and proven customer success.

BJSS has announced it has attained yet another AWS competency – this time focused on its security expertise.

This new competency makes BJSS one of a select group of partners across the whole of Europe and the Middle East that has achieved an AWS Competency in Security.

The AWS Competency Program is a partner program that recognises Amazon Web Services (AWS) consulting partners who have demonstrated best-in-practice technical expertise in specialised topics within the AWS cloud.

BJSS has designed, developed, and built security solutions for organisations such as NHS Digital and the DVSA, as well as renowned financial institutions. Their solutions have been audited for security engineering, operations and automation (DevSecOps) which are some of the core components of any modern security strategy on AWS.

According to Amazon Web Services, organisations that achieve AWS Security Competency have a deep technical expertise and proven customer success securing every stage of cloud adoption, from initial migration through ongoing day-to-day management.

Mohamed Zamzam, Cloud Migrations and Services Lead at BJSS, said: “We are delighted to become one of the select consulting firms that have the AWS Security competency. AWS Partners are measured against a high bar to achieve this AWS Security Competency designation. A validation that showcases our capabilities to deliver best-in-practice security solutions for our clients using AWS services.”

“As the cyber security threat continues to grow, it is more important than ever that organisations protect themselves. Security is a broad topic, and it requires deep evaluation at a granular level to be effective. With this competency, clients should have confidence in BJSS to provide this."

The four AWS Competencies that BJSS have attained are;

  • Government competency
  • DevOps competency
  • Migration competency
  • Security competency

Organisations can now benefit from BJSS Enterprise Cloud — a secure and scalable landing zone accelerator that has helped both public and private sector clients accelerate their journeys to AWS.

For BJSS cloud security offerings and consulting engagements, visit BJSS.com/cloud.
